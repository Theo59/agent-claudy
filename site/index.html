Tes agents Claude Code,
en têtes de Claudy.

Chaque session Claude Code devient une tête de Claudy Focan qui parle, bosse, et te réclame quand elle a besoin de toi. Local, fun, zéro dépendance.

▾ Télécharger l'installeur Voir la démo

macOS · Node ≥ 18 · « Éducation minimum. »

① Dans Claude Code, ça bosse…

claude — agent-claudy

② …et Claudy te le montre en direct — la vraie interface :

agent-claudy

Trois temps, c'est tout

  1. 1 Lance agent-claudy (ou l'app menubar).
  2. 2 Tes sessions Claude Code apparaissent toutes seules.
  3. 3 Clique une tête → la fenêtre de l'agent revient au premier plan.

Ce qu'il sait faire

Essaim de sous-agents

Une mini-tête par sous-agent/workflow : vert = fini, rouge = raté, ambre = en cours, + compteur.

« Y a moyen de tout, tout est négociable. »

Notifs cliquables

Quand un agent te réclame, la notif porte le logo et le clic te ramène à la bonne session.

« Chef, faut prendre une décision ! »

Menubar + fenêtre flottante

Toujours sous la main dans la barre de menus, ou en vignette flottante (raccourci ⌃⌥C).

« T'es tendue comme une crampe. »

Focus de la fenêtre

Un clic sur une tête et VS Code / le terminal de cette session revient devant.

« Remets la petite sœur. »

Réglages intégrés

Tout se configure dans l'UI (⚙) — aucune variable d'environnement à éditer.

« Éducation minimum ! »

Zéro dépendance

Node natif + SSE. Ça se lance en local en deux secondes, sans usine à gaz.

« À la revoyure. »

Installe Claudy

macOS · Node ≥ 18 · 100 % local

Le plus simple

curl -fsSL https://raw.githubusercontent.com/Theo59/agent-claudy/main/install.sh | bash

Récupère le projet, compile les apps macOS, te dit comment lancer.
