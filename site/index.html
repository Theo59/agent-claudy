Questions fréquentes

Qu'est-ce qu'Agent Claudy ?

Agent Claudy est un visualiseur local et open source (licence MIT) pour Claude Code : chaque session active devient une tête pixel-art de Claudy Focan qui parle quand elle travaille, attend quand elle a fini, et passe au rouge quand elle réclame une réponse. Il tourne sur macOS, dans le navigateur, avec ou sans VS Code.

Comment Agent Claudy découvre-t-il mes sessions ?

Agent Claudy découvre automatiquement les sessions Claude Code de la machine : tu lances l'app (barre de menus ou navigateur) et les têtes apparaissent toutes seules, sans configuration. Un clic sur une tête ramène au premier plan la fenêtre VS Code ou le terminal de cette session.

Est-ce gratuit et open source ?

Oui. Agent Claudy est entièrement gratuit et open source sous licence MIT, et son code est public sur GitHub. Il tourne 100 % en local sur ta machine : aucune donnée n'est envoyée à un serveur tiers, et il n'y a ni compte ni abonnement.

Quelles plateformes et prérequis ?

Agent Claudy cible macOS et nécessite Node.js 18 ou plus. L'interface s'ouvre dans le navigateur, avec ou sans VS Code, et des apps natives — barre de menus et fenêtre flottante — complètent l'expérience. L'installation se fait en une commande curl ou via un fichier .dmg.

Faut-il installer des dépendances ?

Non, aucune. Agent Claudy n'a aucune dépendance npm : le serveur utilise uniquement le module http natif de Node et les Server-Sent Events. Il se lance en local en quelques secondes, sans étape de build — d'où le « zéro dépendance ».

Affiche-t-il les sous-agents et workflows ?

Oui. Pour chaque session, Agent Claudy affiche un essaim de mini-têtes — une par sous-agent ou workflow actif — colorées par statut : vert quand c'est fini, rouge en cas d'échec, ambre en cours, avec un compteur de progression pour suivre un workflow en direct.
